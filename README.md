# Lab Exercise Evaluation Web Application

A web application for evaluating laboratory exercises, developed as part of the Electronic and Mobile Commerce course. The system is built with Spring Boot and React, and containerized with Docker for easier setup and deployment.

## Overview

This project is designed to simplify the process of evaluating laboratory exercises through a modern and organized web interface. It enables efficient management of submissions, reviews, and assessment workflows for both students and instructors.

## Tech Stack

- Backend: Spring Boot
- Frontend: React
- Database: PostgreSQL
- Containerization: Docker

## Features

- User-friendly web interface
- Laboratory exercise evaluation and management
- Organized assessment workflow
- Backend and frontend separation
- Docker-based setup for easier development and deployment

## Installation

### Clone the repository

```bash
git clone https://github.com/username/repository.git
cd repository
```

### Run with Docker

```bash
docker-compose up --build
```

### Run manually

#### Backend

```bash
cd backend
./mvnw spring-boot:run
```

#### Frontend

```bash
cd frontend
npm install
npm start
```

## Usage

After starting the application, open the frontend in your browser and use the platform to manage and evaluate laboratory exercises.

## Purpose

The main goal of this project is to provide a practical solution for digital assessment in academic environments, with a focus on simplicity, efficiency, and better organization of laboratory evaluations.
